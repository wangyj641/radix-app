# Monitor
- Front-end app

### Key Skills
- JavaScript, React, JWT, antd, echarts

### Key Features
- Data visualization for sensors
- JWT token login
- Upload CSV 

### Getting Started
- Start server first
- Clone the code and install:
  ```shell
  npm install
  ```

- Start app:
  ```shell
  npm start
  ```

- Check http://localhost:3000/
  Login
  ```shell
  user: admin
  password: 123456
  ```
  
  ### How to test
  ```shell
  npm test
  ```
